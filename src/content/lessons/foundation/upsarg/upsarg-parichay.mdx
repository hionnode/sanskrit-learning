## उपसर्ग क्या है?

**उपसर्ग** (verbal prefix) वे शब्दांश हैं जो **धातु** (verb root) के पहले जुड़कर उसके अर्थ को **परिवर्तित, विस्तृत या पूर्णतया बदल** देते हैं।

> **उपसर्गेण धात्वर्थो बलादन्यत्र नीयते।**
> **प्रहाराहारसंहारविहारपरिहारवत्॥**

अर्थात्: उपसर्ग से धातु का अर्थ बलपूर्वक अन्यत्र ले जाया जाता है — जैसे 'हृ' धातु से: प्रहार, आहार, संहार, विहार, परिहार — सब भिन्न अर्थ!

## एक धातु, अनेक उपसर्ग — अनेक अर्थ

### 'हृ' धातु (ले जाना / to carry)

| उपसर्ग + हृ | शब्द | अर्थ |
|-------------|------|------|
| **प्र** + हृ | प्रहार | आगे ले जाना → मारना/प्रहार |
| **आ** + हृ | आहार | ओर ले आना → भोजन/आहार |
| **सम्** + हृ | संहार | साथ समेटना → संहार/नाश |
| **वि** + हृ | विहार | विशेष रूप ले जाना → विहार/भ्रमण |
| **परि** + हृ | परिहार | चारों ओर ले जाना → बचाव/त्याग |
| **उप** + हृ | उपहार | पास ले जाना → भेंट/उपहार |

### 'गम्' धातु (जाना / to go)

| उपसर्ग + गम् | रूप | अर्थ |
|-------------|------|------|
| मूल **गम्** | गच्छति | जाता है |
| **आ** + गम् | आगच्छति | आता है |
| **प्र** + गम् | प्रगच्छति | आगे बढ़ता है |
| **उद्** + गम् | उद्गच्छति | ऊपर उठता है |
| **अव** + गम् | अवगच्छति | समझता है |
| **सम्** + गम् | संगच्छते | मिलता है |
| **निर्** + गम् | निर्गच्छति | बाहर निकलता है |
| **अनु** + गम् | अनुगच्छति | पीछे जाता है |
| **प्रति** + गम् | प्रतिगच्छति | लौटता है |
| **उप** + गम् | उपगच्छति | पास जाता है |

## उपसर्ग कैसे अर्थ बदलते हैं?

उपसर्ग तीन प्रकार से प्रभाव डालते हैं:

### 1. अर्थ में दिशा-परिवर्तन

> गम् (जाना) → **आ**गम् (आना) — दिशा उलट गई

### 2. अर्थ में विस्तार / सूक्ष्म भेद

> गम् (जाना) → **प्र**गम् (प्रगति करना) — 'आगे' का विस्तार

### 3. अर्थ में पूर्ण परिवर्तन

> गम् (जाना) → **अव**गम् (समझना) — 'जाना' से 'समझना' — पूर्णतया नया अर्थ!

## 'कृ' धातु (करना) — उपसर्गों के साथ

| उपसर्ग + कृ | शब्द | अर्थ |
|-------------|------|------|
| **प्र** + कृ | प्रकृति | मूल स्वभाव |
| **सम्** + कृ | संस्कार | भली प्रकार करना, शुद्धि |
| **वि** + कृ | विकार | विकृत करना, बिगाड़ |
| **उप** + कृ | उपकार | सहायता करना |
| **अप** + कृ | अपकार | बुरा करना, हानि |
| **परि** + कृ | परिकर | चारों ओर करना, तैयारी |
| **आ** + कृ | आकार | रूप देना |
| **निर्** + कृ | निराकार | रूप रहित |

## आगामी पाठों में

आगामी पाठों में हम उपसर्गों को विस्तार से पढ़ेंगे:

1. **प्र, परा, अप** — आगे, दूर, हटकर
2. **सम्, अनु, अव** — साथ, पीछे, नीचे
3. **निस्, नि, उद्** — बाहर, भीतर, ऊपर
4. **अभि, अधि, प्रति** — सामने, ऊपर, वापस

## याद रखें

1. **उपसर्ग** = धातु से पहले जुड़ने वाले शब्दांश — अर्थ बदलते हैं
2. **22 उपसर्ग** (पाणिनि): प्र, परा, अप, सम्, अनु, अव, निस्, दुस्, वि, आ, नि, अधि, अपि, अति, सु, उद्, अभि, प्रति, परि, उप
3. एक ही धातु + भिन्न उपसर्ग = **भिन्न अर्थ** (हृ → प्रहार, आहार, संहार, विहार, उपहार)
4. उपसर्ग **दिशा बदल** सकते हैं, **विस्तार** कर सकते हैं, या **पूर्णतया नया अर्थ** दे सकते हैं
5. अनेक हिन्दी शब्द (प्रगति, संस्कार, विज्ञान, उपहार) संस्कृत **उपसर्ग + धातु** से बने हैं
