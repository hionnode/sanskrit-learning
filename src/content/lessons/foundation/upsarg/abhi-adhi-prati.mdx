## अभि, अधि, प्रति — तीन उपसर्ग

इस पाठ में तीन उपसर्ग: **अभि** (ओर/सामने), **अधि** (ऊपर), **प्रति** (वापस/प्रत्येक)।

## 1. अभि (ओर / towards / before)

**'अभि'** का मूल अर्थ है **'ओर'** (towards), **'सामने'** (before/in front of), या **'तीव्रता'** (intensely)।

### अभि के विभिन्न अर्थ-छायाएँ

| अर्थ-छाया | उदाहरण | विच्छेद | अर्थ |
|-----------|---------|---------|------|
| **ओर/सामने** | अभिगच्छति | अभि + गम् | ओर जाता है / पास जाता है |
| **तीव्रता** | अभिलाषा | अभि + लष् | तीव्र इच्छा |
| **सामने (performance)** | अभिनयः | अभि + नी | सामने ले जाना → अभिनय |
| **चारों ओर** | अभिषेकः | अभि + सिच् | चारों ओर सींचना → राज्याभिषेक |

### प्रमुख शब्द — 'अभि' उपसर्ग

| शब्द | धातु | अर्थ |
|------|------|------|
| **अभिनन्दनम्** | अभि + नन्द् | सामने आकर स्वागत, welcome |
| **अभिनयः** | अभि + नी | सामने ले जाना → नाट्य, acting |
| **अभिलाषा** | अभि + लष् | तीव्र इच्छा, desire |
| **अभिमानः** | अभि + मन् | स्वयं के प्रति तीव्र मान, pride |
| **अभिषेकः** | अभि + सिच् | चारों ओर सींचना, coronation |
| **अभिज्ञः** | अभि + ज्ञा | सामने जानना → जानकार, expert |
| **अभिप्रायः** | अभि + प्र + इ | ओर जाना → अर्थ/मत, meaning/opinion |
| **अभ्यासः** | अभि + अस् | ओर बार-बार → अभ्यास, practice |

### वाक्यों में प्रयोग

> **श्रेष्ठिनः अतिथेः अभिनन्दनं कुर्वन्ति।** = श्रेष्ठीजन अतिथि का अभिनन्दन करते हैं।
> **नटः रङ्गमञ्चे अभिनयं करोति।** = अभिनेता रंगमंच पर अभिनय करता है।
> **अभ्यासेन सर्वं सिध्यति।** = अभ्यास से सब सिद्ध होता है।

## 2. अधि (ऊपर / over / upon)

**'अधि'** का मूल अर्थ है **'ऊपर'** (over/upon/above) या **'अतिरिक्त'** (additional/superior)।

### अधि के विभिन्न अर्थ-छायाएँ

| अर्थ-छाया | उदाहरण | विच्छेद | अर्थ |
|-----------|---------|---------|------|
| **ऊपर/पर** | अधिकारः | अधि + कृ | ऊपर से करना → अधिकार |
| **ऊपर (पढ़ना)** | अध्ययनम् | अधि + इ | ऊपर जाना → पढ़ना |
| **प्रमुख/मुखिया** | अधिपतिः | अधि + पति | ऊपर का पति → स्वामी |
| **अतिरिक्त** | अधिकम् | अधि + क | ऊपर → अधिक |

### सन्धि: अधि + इ → अध्य

जब 'अधि' के बाद स्वर 'इ' आता है:
- अधि + इ = **अध्य** (इ → य)
- अध्य + यनम् = **अध्ययनम्** (पढ़ना)
- अध्य + अपकः = **अध्यापकः** (पढ़ाने वाला)

### प्रमुख शब्द — 'अधि' उपसर्ग

| शब्द | धातु | अर्थ |
|------|------|------|
| **अधिकारः** | अधि + कृ | ऊपर नियन्त्रण, authority |
| **अध्ययनम्** | अधि + इ | पढ़ना, study |
| **अध्यापकः** | अधि + इ (causal) | पढ़ाने वाला, teacher |
| **अधिपतिः** | अधि + पति | स्वामी/मुखिया, lord |
| **अधिगमः** | अधि + गम् | प्राप्ति/ज्ञान, attainment |
| **अधिवासः** | अधि + वस् | ऊपर रहना → वास, dwelling |
| **अधिष्ठानम्** | अधि + स्था | ऊपर खड़ा → आधार, foundation |

### वाक्यों में प्रयोग

> **छात्रः संस्कृतस्य अध्ययनं करोति।** = छात्र संस्कृत का अध्ययन करता है।
> **अध्यापकः पाठं पाठयति।** = अध्यापक पाठ पढ़ाता है।
> **राजा राज्यस्य अधिपतिः अस्ति।** = राजा राज्य का अधिपति है।

## 3. प्रति (वापस / back / against / each)

**'प्रति'** का मूल अर्थ है **'वापस'** (back), **'विपरीत'** (against), या **'प्रत्येक'** (each/every)।

### प्रति के विभिन्न अर्थ-छायाएँ

| अर्थ-छाया | उदाहरण | विच्छेद | अर्थ |
|-----------|---------|---------|------|
| **वापस** | प्रतिगच्छति | प्रति + गम् | वापस जाता है |
| **प्रत्युत्तर** | प्रतिक्रिया | प्रति + कृ | वापस करना → प्रतिक्रिया |
| **विपरीत** | प्रतिकूलः | प्रति + कूल | विपरीत किनारा → प्रतिकूल |
| **प्रत्येक** | प्रतिदिनम् | प्रति + दिन | प्रत्येक दिन |
| **प्रतिज्ञा** | प्रतिज्ञा | प्रति + ज्ञा | सामने जानकर → वचन, promise |

### प्रमुख शब्द — 'प्रति' उपसर्ग

| शब्द | धातु | अर्थ |
|------|------|------|
| **प्रतिक्रिया** | प्रति + कृ | वापस करना, reaction |
| **प्रतिदिनम्** | प्रति + दिन | हर दिन, daily |
| **प्रतिज्ञा** | प्रति + ज्ञा | वचन, promise |
| **प्रतिमा** | प्रति + मा | अनुसार मापना → मूर्ति, statue |
| **प्रतिबिम्बम्** | प्रति + बिम्ब | वापस बिम्ब → प्रतिबिम्ब, reflection |
| **प्रतीक्षा** | प्रति + ईक्ष् | सामने देखना → प्रतीक्षा, waiting |
| **प्रतिकूलः** | प्रति + कूल | विपरीत किनारा → प्रतिकूल, adverse |
| **प्रतिवादी** | प्रति + वद् | वापस बोलने वाला, defendant |

### वाक्यों में प्रयोग

> **सैनिकः शत्रोः प्रतिक्रियां करोति।** = सैनिक शत्रु की प्रतिक्रिया करता है।
> **सः प्रतिदिनम् व्यायामं करोति।** = वह प्रतिदिन व्यायाम करता है।
> **राजा प्रतिज्ञां करोति।** = राजा प्रतिज्ञा करता है।

## तुलना — अभि, अधि, प्रति

| उपसर्ग | दिशा | भाव | उदाहरण |
|---------|------|-----|---------|
| **अभि** | → ओर/सामने | तीव्रता, सम्मुख | अभिनन्दन, अभिनय, अभिलाषा |
| **अधि** | ↑ ऊपर | अधिकार, श्रेष्ठता | अधिकार, अध्ययन, अधिपति |
| **प्रति** | ← वापस/प्रत्येक | प्रत्युत्तर, विपरीत | प्रतिक्रिया, प्रतिदिन, प्रतिज्ञा |

## अभ्यास — 'गम्' धातु + तीनों उपसर्ग

| उपसर्ग + गम् | रूप | अर्थ |
|-------------|------|------|
| अभि + गम् | **अभिगच्छति** | ओर/पास जाता है |
| अधि + गम् | **अधिगच्छति** | ऊपर जाना → प्राप्त करता है |
| प्रति + गम् | **प्रतिगच्छति** | वापस जाता है / लौटता है |

## सभी उपसर्गों का सारांश (अब तक)

| उपसर्ग | मूल अर्थ | प्रमुख शब्द |
|---------|----------|------------|
| **प्र** | आगे | प्रगति, प्रवेश |
| **परा** | दूर/पूर्णतया | पराजय, पराक्रम |
| **अप** | हटकर/बुरा | अपमान, अपकार |
| **सम्** | साथ/भली प्रकार | संगम, संस्कार |
| **अनु** | पीछे/अनुसार | अनुभव, अनुवाद |
| **अव** | नीचे | अवतार, अवगमन |
| **निस्/निर्** | बाहर/बिना | निर्गम, निर्भय |
| **नि** | नीचे/भीतर | निवास, नियम |
| **उद्** | ऊपर | उद्गम, उत्थान |
| **अभि** | ओर/सामने | अभिनन्दन, अभिनय |
| **अधि** | ऊपर/अतिरिक्त | अधिकार, अध्ययन |
| **प्रति** | वापस/प्रत्येक | प्रतिक्रिया, प्रतिदिन |

## याद रखें

1. **अभि** = ओर/सामने/तीव्रता — अभिनन्दन, अभिनय, अभिलाषा, अभ्यास
2. **अधि** = ऊपर/अतिरिक्त — अधिकार, अध्ययन, अध्यापक, अधिपति
3. **प्रति** = वापस/विपरीत/प्रत्येक — प्रतिक्रिया, प्रतिदिन, प्रतिज्ञा, प्रतिबिम्ब
4. **प्रति** के तीन अर्थ: वापस (प्रतिगम्), विपरीत (प्रतिकूल), प्रत्येक (प्रतिदिन)
5. **अधि + इ** = अध्य (सन्धि) → अध्ययन, अध्यापक
