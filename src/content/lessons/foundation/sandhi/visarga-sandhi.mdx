## विसर्ग सन्धि क्या है?

**विसर्ग** (:) शब्द के अन्त में आने वाला एक विशेष वर्ण है। यह मूलतः **स्** या **र्** का रूप है। जब विसर्ग के बाद कोई स्वर या व्यञ्जन आता है, तो विसर्ग में परिवर्तन होता है — इसे **विसर्ग सन्धि** कहते हैं।

विसर्ग सन्धि के मुख्य नियम:

1. **उत्व** — अः + घोष वर्ण/स्वर → ओ
2. **सत्व** — ः + त/थ → स्
3. **शत्व** — ः + च/छ → श्
4. **षत्व** — ः + ट/ठ → ष्
5. **रुत्व** — ः + स्वर → र्
6. **विसर्ग लोप** — आः + घोष → आ

## 1. उत्व — अः + घोष व्यञ्जन/स्वर

जब **अः** के बाद **घोष** व्यञ्जन (ग, घ, ज, झ, ड, ढ, द, ध, ब, भ, य, र, ल, व, ह, न, म) या कोई **स्वर** (अ को छोड़कर) आता है:

| मूल | सन्धि | नियम |
|-----|--------|------|
| रामः + गच्छति | **रामो गच्छति** | अः + ग → ओ |
| शिवः + वदति | **शिवो वदति** | अः + व → ओ |
| बालकः + धावति | **बालको धावति** | अः + ध → ओ |
| रामः + इच्छति | **रामो इच्छति** | अः + इ → ओ |

> **विशेष:** अः + अ → ओ + ऽ (पूर्वरूप)। जैसे: रामः + अवदत् = रामोऽवदत्

## 2. विसर्ग लोप — आः + घोष

जब **आः** के बाद घोष व्यञ्जन या स्वर आता है तो विसर्ग **लुप्त** हो जाता है:

| मूल | सन्धि | नियम |
|-----|--------|------|
| देवाः + नमन्ति | **देवा नमन्ति** | आः + न → आ |
| बालिकाः + गच्छन्ति | **बालिका गच्छन्ति** | आः + ग → आ |
| देवाः + अत्र | **देवा अत्र** | आः + अ → आ |

## 3. सत्व — विसर्ग + त-वर्ग (त, थ)

विसर्ग **त/थ** से पहले **स्** बन जाता है:

| मूल | सन्धि | नियम |
|-----|--------|------|
| मनः + तापः | **मनस्तापः** | ः + त → स्त |
| नमः + ते | **नमस्ते** | ः + त → स्त |
| तपः + ताप | **तपस्ताप** | ः + त → स्त |

## 4. शत्व — विसर्ग + च-वर्ग (च, छ)

विसर्ग **च/छ** से पहले **श्** बन जाता है:

| मूल | सन्धि | नियम |
|-----|--------|------|
| रामः + चरति | **रामश्चरति** | ः + च → श्च |
| हरिः + चलति | **हरिश्चलति** | ः + च → श्च |
| तपः + चर्या | **तपश्चर्या** | ः + च → श्च |

## 5. षत्व — विसर्ग + ट-वर्ग (ट, ठ)

विसर्ग **ट/ठ** से पहले **ष्** बन जाता है:

| मूल | सन्धि | नियम |
|-----|--------|------|
| धनुः + टङ्कारः | **धनुष्टङ्कारः** | ः + ट → ष्ट |
| निः + ठुरः | **निष्ठुरः** | ः + ठ → ष्ठ |

## 6. रुत्व — विसर्ग + स्वर (अ/आ को छोड़कर)

कुछ विशेष स्थितियों में विसर्ग **र्** बन जाता है:

| मूल | सन्धि | नियम |
|-----|--------|------|
| निः + अर्थकम् | **निरर्थकम्** | ः + अ → र |
| दुः + आत्मा | **दुरात्मा** | ः + आ → र |
| निः + उपमम् | **निरुपमम्** | ः + उ → र |

## सन्धि-विच्छेद सारणी

| सन्धि शब्द | विच्छेद | नियम |
|------------|---------|------|
| रामो गच्छति | रामः + गच्छति | उत्व (अः+घोष→ओ) |
| नमस्ते | नमः + ते | सत्व (ः+त→स्त) |
| हरिश्चन्द्रः | हरिः + चन्द्रः | शत्व (ः+च→श्च) |
| दुष्कर्म | दुः + कर्म | षत्व (ः+क→ष्क) |
| निरर्थकम् | निः + अर्थकम् | रुत्व (ः+स्वर→र) |
| मनस्तापः | मनः + तापः | सत्व (ः+त→स्त) |

## मूल पाठ में प्रयोग

इन वाक्यों में विसर्ग सन्धि पहचानें:

> **रामो वनं गच्छति। नमस्ते गुरवे। तपश्चर्या निष्फला न भवति।**

| सन्धि शब्द | विच्छेद | नियम |
|------------|---------|------|
| रामो | रामः + (व) | उत्व (अः+घोष→ओ) |
| नमस्ते | नमः + ते | सत्व (ः+त→स्त) |
| तपश्चर्या | तपः + चर्या | शत्व (ः+च→श्च) |
| निष्फला | निः + फला | षत्व (ः+फ→ष्फ) |

**अनुवाद:** राम वन जाता है। गुरु को नमस्कार। तपश्चर्या निष्फल नहीं होती।

## याद रखें

1. **उत्व** — अः + घोष वर्ण → ओ (रामः+गच्छति = रामो गच्छति)
2. **सत्व** — ः + त/थ → स् (नमः+ते = नमस्ते)
3. **शत्व** — ः + च/छ → श् (तपः+चर्या = तपश्चर्या)
4. **षत्व** — ः + ट/ठ → ष् (निः+ठुर = निष्ठुर)
5. **रुत्व** — ः + स्वर → र् (निः+अर्थ = निरर्थ)
6. **विसर्ग लोप** — आः + घोष → आ (देवाः+नमन्ति = देवा नमन्ति)
7. "नमस्ते" में विसर्ग सन्धि है — नमः + ते!
