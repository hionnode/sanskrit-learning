## लृट् लकार — भविष्यत् काल

पिछले पाठ में हमने **लट् लकार** (वर्तमान काल) सीखा। अब **लृट् लकार** (भविष्यत् काल / future tense) सीखें।

## भविष्यत् काल कैसे बनता है?

धातु के अंश में **-इष्य-** जोड़कर, फिर पुरुष-वचन प्रत्यय लगाकर:

**धातु अंश + इष्य + पुरुष प्रत्यय**

| धातु | अंश | + इष्य | उदाहरण (प्र.पु.ए.) |
|------|------|--------|-------------------|
| गम् | गम- | गमिष्य- | गम् + इष्य + ति = **गमिष्यति** |
| पठ् | पठ- | पठिष्य- | पठ् + इष्य + ति = **पठिष्यति** |
| भू | भव- | भविष्य- | भव् + इष्य + ति = **भविष्यति** |

## गम् धातु — लृट् लकार (9 रूप)

| | एकवचन | द्विवचन | बहुवचन |
|--|--------|---------|--------|
| **प्रथम पुरुष** | गमिष्यति | गमिष्यतः | गमिष्यन्ति |
| **मध्यम पुरुष** | गमिष्यसि | गमिष्यथः | गमिष्यथ |
| **उत्तम पुरुष** | गमिष्यामि | गमिष्यावः | गमिष्यामः |

## पठ् धातु — लृट् लकार

| | एकवचन | द्विवचन | बहुवचन |
|--|--------|---------|--------|
| **प्रथम पुरुष** | पठिष्यति | पठिष्यतः | पठिष्यन्ति |
| **मध्यम पुरुष** | पठिष्यसि | पठिष्यथः | पठिष्यथ |
| **उत्तम पुरुष** | पठिष्यामि | पठिष्यावः | पठिष्यामः |

## भू धातु — लृट् लकार

| | एकवचन | द्विवचन | बहुवचन |
|--|--------|---------|--------|
| **प्रथम पुरुष** | भविष्यति | भविष्यतः | भविष्यन्ति |
| **मध्यम पुरुष** | भविष्यसि | भविष्यथः | भविष्यथ |
| **उत्तम पुरुष** | भविष्यामि | भविष्यावः | भविष्यामः |

## लट् बनाम लृट् — तुलना

| पुरुष + वचन | गम् (लट्) | गम् (लृट्) |
|-------------|-----------|-----------|
| प्र.पु.ए. | गच्छ**ति** | गमिष्य**ति** |
| प्र.पु.द्वि. | गच्छ**तः** | गमिष्य**तः** |
| प्र.पु.ब. | गच्छ**न्ति** | गमिष्य**न्ति** |
| म.पु.ए. | गच्छ**सि** | गमिष्य**सि** |
| म.पु.द्वि. | गच्छ**थः** | गमिष्य**थः** |
| म.पु.ब. | गच्छ**थ** | गमिष्य**थ** |
| उ.पु.ए. | गच्छा**मि** | गमिष्या**मि** |
| उ.पु.द्वि. | गच्छा**वः** | गमिष्या**वः** |
| उ.पु.ब. | गच्छा**मः** | गमिष्या**मः** |

ध्यान दें — **पुरुष-वचन प्रत्यय** (ति, तः, न्ति, सि...) **दोनों लकारों में एक जैसे** हैं। केवल बीच का अंश बदलता है: गच्छ- (लट्) → गमिष्य- (लृट्)।

## उदाहरण वाक्य

| संस्कृत | अर्थ |
|---------|------|
| **बालकः श्वः पठिष्यति** | लड़का कल पढ़ेगा |
| **बालकौ श्वः पठिष्यतः** | दो लड़के कल पढ़ेंगे |
| **बालकाः श्वः पठिष्यन्ति** | सब लड़के कल पढ़ेंगे |
| **अहं श्वः गमिष्यामि** | मैं कल जाऊँगा |
| **त्वं कदा पठिष्यसि?** | तू कब पढ़ेगा? |

## मूल पाठ में प्रयोग

> **बालकः श्वः विद्यालयं गमिष्यति। तत्र सः संस्कृतं पठिष्यति। सायं सः गृहम् आगमिष्यति।**

| शब्द | अर्थ | व्याकरण |
|------|------|---------|
| बालकः | लड़का | प्रथमा एकवचन |
| श्वः | कल | काल-वाचक अव्यय |
| विद्यालयम् | विद्यालय को | द्वितीया विभक्ति |
| गमिष्यति | (वह) जाएगा | गम् धातु, लृट्, प्र.पु.ए. |
| संस्कृतम् | संस्कृत को | द्वितीया विभक्ति |
| पठिष्यति | (वह) पढ़ेगा | पठ् धातु, लृट्, प्र.पु.ए. |
| आगमिष्यति | (वह) आएगा | आगम् धातु, लृट्, प्र.पु.ए. |

**अनुवाद:** लड़का कल विद्यालय जाएगा। वहाँ वह संस्कृत पढ़ेगा। शाम को वह घर आएगा।

## याद रखें

1. **लृट् लकार** = भविष्यत् काल। धातु अंश + **-इष्य-** + पुरुष प्रत्यय
2. पुरुष-वचन प्रत्यय **लट् और लृट् में एक जैसे** हैं
3. गम् → गमिष्य-, पठ् → पठिष्य-, भू → भविष्य-
4. यही '-इष्य-' चिह्न भविष्यत् काल पहचानने का सबसे आसान तरीका है
