## स्वर सन्धि के विशेष नियम

आधारभूत पाठ में हमने सवर्ण दीर्घ, गुण और वृद्धि सन्धि पढ़ी। अब कुछ और महत्त्वपूर्ण स्वर सन्धि के नियम पढ़ेंगे।

## 1. यण् सन्धि

जब **इ/ई, उ/ऊ, ऋ/ॠ** के बाद कोई **असमान स्वर** (भिन्न स्वर) आता है, तो:

| पूर्व स्वर | आदेश | सूत्र |
|-----------|-------|-------|
| इ / ई | **य्** | इको यणचि |
| उ / ऊ | **व्** | इको यणचि |
| ऋ / ॠ | **र्** | इको यणचि |

### उदाहरण

| सन्धि से पहले | सन्धि | नियम |
|---------------|--------|------|
| इति + आह | **इत्याह** | इ + आ → य् + आ |
| अति + अन्तम् | **अत्यन्तम्** | इ + अ → य् + अ |
| प्रति + एकम् | **प्रत्येकम्** | इ + ए → य् + ए |
| मधु + अरिः | **मध्वरिः** | उ + अ → व् + अ |
| अनु + एषणम् | **अन्वेषणम्** | उ + ए → व् + ए |
| गुरु + आदेशः | **गुर्वादेशः** | उ + आ → व् + आ |
| पितृ + आदेशः | **पित्रादेशः** | ऋ + आ → र् + आ |
| मातृ + अर्थम् | **मात्रर्थम्** | ऋ + अ → र् + अ |

## 2. अयादि सन्धि

जब **ए, ऐ, ओ, औ** (संयुक्त/वृद्धि स्वर) के बाद कोई स्वर आता है:

| पूर्व स्वर | आदेश | सूत्र |
|-----------|-------|-------|
| ए | **अय्** | एचोऽयवायावः |
| ऐ | **आय्** | एचोऽयवायावः |
| ओ | **अव्** | एचोऽयवायावः |
| औ | **आव्** | एचोऽयवायावः |

### उदाहरण

| सन्धि से पहले | सन्धि | नियम |
|---------------|--------|------|
| ने + अनम् | **नयनम्** | ए → अय् |
| नै + अकः | **नायकः** | ऐ → आय् |
| भो + अनम् | **भवनम्** | ओ → अव् |
| पौ + अकः | **पावकः** | औ → आव् |
| शे + अनम् | **शयनम्** | ए → अय् |
| गै + अकः | **गायकः** | ऐ → आय् |
| श्रो + अणम् | **श्रवणम्** | ओ → अव् |

> **ध्यान दें:** अयादि सन्धि प्रायः धातुओं और प्रत्ययों के बीच होती है, शब्दों के बीच में कम।

## 3. पूर्वरूप सन्धि

पदान्त **ए/ओ** के बाद **अ** आने पर, 'अ' लुप्त हो जाता है और पूर्व स्वर (ए/ओ) बना रहता है:

> **सो + अपि** → **सोऽपि** (वह भी)
> **तेन + अपि** → ~~तेनापि~~ (यह सवर्ण दीर्घ नहीं — 'ए' + 'अ')

वस्तुतः पूर्वरूप सन्धि पदान्त ए/ओ + अ पर लागू होती है। लेखन में अवग्रह (ऽ) लगाया जाता है:
- सः + अपि → **सोऽपि**
- एषः + अपि → **एषोऽपि**

## 4. पररूप सन्धि

उपसर्ग के अन्त में **अ/आ** हो और धातु **ए/ओ** से शुरू हो, तो पररूप (बाद वाला) बचता है:

> **प्र + एजते** → **प्रेजते** (न कि प्रैजते)
> **उप + ओषति** → **उपोषति**

## 5. प्रकृतिभाव (सन्धि का अभाव)

कुछ स्थितियों में सन्धि **नहीं** होती:

| स्थिति | उदाहरण |
|--------|---------|
| **द्विवचन ई/ऊ** के बाद | कवी इमौ (कव्यिमौ नहीं) |
| **अमी** सर्वनाम में | अमी इह |
| **सम्बोधन** के बाद | हे अर्जुन (हार्जुन नहीं) |

## सारांश तालिका

| सन्धि | नियम | उदाहरण |
|-------|------|---------|
| **यण्** | इ/ई→य्, उ/ऊ→व्, ऋ→र् (+ असमान स्वर) | इत्याह, मध्वरिः, पित्रादेशः |
| **अयादि** | ए→अय्, ऐ→आय्, ओ→अव्, औ→आव् (+ स्वर) | नयनम्, नायकः, भवनम्, पावकः |
| **पूर्वरूप** | पदान्त ए/ओ + अ → ए/ओ (ऽ) | सोऽपि, एषोऽपि |
| **प्रकृतिभाव** | सन्धि नहीं होती | कवी इमौ |

## याद रखें

1. **यण् सन्धि**: इ→य्, उ→व्, ऋ→र् + असमान स्वर
2. **अयादि**: ए→अय्, ऐ→आय्, ओ→अव्, औ→आव् + स्वर
3. **पूर्वरूप**: ए/ओ + अ → ए/ओ + ऽ (अवग्रह)
4. **प्रकृतिभाव**: द्विवचन -ई/-ऊ, सम्बोधन आदि में सन्धि नहीं
5. यण् सन्धि सर्वाधिक प्रयुक्त है — इसे अवश्य याद रखें
