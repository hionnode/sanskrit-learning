## विसर्ग सन्धि के विशेष नियम

विसर्ग (ः) संस्कृत में अत्यन्त सामान्य है और इसकी सन्धि के अनेक नियम हैं। आधारभूत पाठ के आगे अब विस्तार से समझेंगे।

## 1. रुत्व — विसर्ग → उ (ओ)

जब **अः** (अ + विसर्ग) के बाद **घोष वर्ण** (अ को छोड़कर अन्य स्वर, वर्ग का तृतीय-चतुर्थ-पञ्चम, य, र, ल, व, ह) आए:

**विसर्ग → उ** → फिर **अ + उ = ओ** (गुण सन्धि)

| सन्धि से पहले | सन्धि | नियम |
|---------------|--------|------|
| रामः + गच्छति | **रामो गच्छति** | अः + ग → ओ + ग |
| बालः + धावति | **बालो धावति** | अः + ध → ओ + ध |
| देवः + इच्छति | **देवेच्छति** | अः + इ → ओ + इ → ए |
| शिवः + उवाच | **शिवोवाच** | अः + उ → ओ + उ → ओ |

> **विशेष:** जब अः + अ → ओ + अ → **ओऽ** (पूर्वरूप + अवग्रह)
> - रामः + अस्ति = **रामोऽस्ति**
> - शिवः + अर्जुनम् = **शिवोऽर्जुनम्**

## 2. विसर्ग → श्, ष्, स्

जब विसर्ग के बाद **अघोष ऊष्म** या **तवर्ग/चवर्ग/टवर्ग** के अघोष वर्ण आएँ:

| विसर्ग + | → | उदाहरण |
|----------|---|---------|
| **श / चवर्ग** | **श्** | रामः + शेते = रामश्शेते |
| **ष / टवर्ग** | **ष्** | धनुः + टङ्कारः = धनुष्टङ्कारः |
| **स / तवर्ग** | **स्** | नमः + ते = नमस्ते |

### उदाहरण

| सन्धि से पहले | सन्धि | नियम |
|---------------|--------|------|
| नमः + ते | **नमस्ते** | ः + त → स् + त |
| मनः + ताप | **मनस्तापः** | ः + त → स् + त |
| रामः + च | **रामश्च** | ः + च → श् + च |
| हरिः + चरति | **हरिश्चरति** | ः + च → श् + च |
| चक्षुः + ष्ठीवनम् | **चक्षुष्ष्ठीवनम्** | ः + ष → ष् + ष |

## 3. विसर्ग → र्

कुछ शब्दों में विसर्ग मूलतः **र्** से आया होता है। जब इसके बाद घोष वर्ण आए:

| सन्धि से पहले | सन्धि | नियम |
|---------------|--------|------|
| पुनः + आगमनम् | **पुनरागमनम्** | ः → र् (मूल र्) |
| अन्तः + गतम् | **अन्तर्गतम्** | ः → र् |
| प्रातः + एव | **प्रातरेव** | ः → र् |
| बहिः + गतः | **बहिर्गतः** | ः → र् |

## 4. विसर्ग लोप

कुछ स्थितियों में विसर्ग लुप्त हो जाता है:

### आः + घोष → आ (विसर्ग लुप्त)
| सन्धि से पहले | सन्धि |
|---------------|--------|
| देवाः + गच्छन्ति | **देवा गच्छन्ति** |
| बालाः + नमन्ति | **बाला नमन्ति** |
| रामाः + अत्र | **रामा अत्र** |

## 5. विसर्ग अपरिवर्तित

जब विसर्ग के बाद **क, ख, प, फ** (कवर्ग/पवर्ग अघोष अल्पप्राण/महाप्राण) आए, तो विसर्ग यथावत् रहता है:

| उदाहरण |
|---------|
| रामः + करोति → **रामः करोति** (विसर्ग यथावत्) |
| मनः + पूतम् → **मनःपूतम्** |
| तपः + फलम् → **तपःफलम्** |

## गीता/सुभाषित से उदाहरण

> **धर्मक्षेत्रे कुरुक्षेत्रे समवेता युयुत्सवः।** (गीता 1.1)
> यहाँ 'समवेताः' + 'युयुत्सवः' = समवेता युयुत्सवः (आः + य → आ + य)

> **नमस्ते अस्तु भगवन्।**
> नमः + ते = नमस्ते (ः + त → स् + त)

## सारांश तालिका

| स्थिति | परिवर्तन | उदाहरण |
|--------|----------|---------|
| अः + घोष (अ के अतिरिक्त) | → **ओ** | रामो गच्छति |
| अः + अ | → **ओऽ** | रामोऽस्ति |
| ः + श/चवर्ग | → **श्** | रामश्च |
| ः + ष/टवर्ग | → **ष्** | धनुष्टङ्कारः |
| ः + स/तवर्ग | → **स्** | नमस्ते |
| ः (मूल र्) + घोष | → **र्** | पुनरागमनम् |
| आः + घोष | → **आ** (विसर्ग लुप्त) | देवा गच्छन्ति |
| ः + क/ख/प/फ | → **ः** (यथावत्) | मनःपूतम् |

## याद रखें

1. **अः + घोष → ओ** — सबसे सामान्य नियम (रामो गच्छति)
2. **ः + त/स → स्** — नमस्ते, मनस्ताप
3. **ः + च/श → श्** — रामश्च, हरिश्चरति
4. **ः + क/प → ः** — विसर्ग नहीं बदलता
5. विसर्ग सन्धि को जानने से **गीता-पाठ** सरल हो जाता है
